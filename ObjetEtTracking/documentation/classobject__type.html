Description détaillée

Objet de manipulation d'un fichier d'extension "obj".

Permet de charger et afficher un fichier "obj". Seuls les sommets, faces, normales et "texture maps" sont récupérés. Le chargement s'effectue vers les types (=struct) vertex_type, normcoord_type, polygon_type, mapcoord_type. L'affichage OpenGL peut être effectué directement grâce à la méthode "render()" (sans se préoccuper des types précédents).
